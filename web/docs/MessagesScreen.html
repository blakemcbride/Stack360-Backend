Summary: The messages screen allows users to send and receive messages with other users of the system. There is a tab for messages received and a tab for messages sent. Controls of Interest: Using the preferences button, users can opt to have messages copied to them via email and specify the email addresses to send to. Selecting the Filter button allows users to restrict the view to search for particular messages. Select the new button to make a new message. Select the view button to view a message. Select Reply to answer a message. Select Delete to remove a message from your list. The message will remain in the database.
